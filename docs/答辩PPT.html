项目背景与意义
为什么做这个项目？

背景：

  • 现代人时间管理需求日益增长
  • 传统日历/待办应用缺乏深度时间分析
  • 多设备数据同步困难

意义：

  • 帮助用户可视化时间分配，提升效率
  • 支持离线（隐私）优先 + 云端同步双保险
  • 跨平台桌面应用，随时随地记录

目标用户：

  • 需要时间管理的上班族
  • 需要追踪学习时间的学生
  • 希望了解时间分配的个人用户
需求分析 - 核心功能
六大功能模块
1. 时间块规划 → 拖拽式创建，专注单一任务
2. 任务追踪 → 实际用时记录，计划vs执行对比
3. 统计报表 → 日/周/月时间流向可视化
4. 双线同步 → WebDAV + 云端账号双保险
5. 离线优先 → 本地存储，有网自动同步
6. 账号系统 → 用户注册/登录/数据隔离
优先级划分
P0（核心）
时间块CRUD、日视图、用户登录
P1（重要）
周视图、月视图、WebDAV同步、分类管理
P2（增强）
自动同步、数据备份、主题切换
