# TimeBlock 时间块记录器 - 五人团队分工与协作方案

## 📋 项目概述

**项目名称**: TimeBlock 时间块记录器  
**技术框架**: Electron + Vue/React + Node.js  
**数据库**: MySQL  
**存储同步**: WebDAV (坚果云/NextCloud/自建服务器)  
**协作平台**: GitHub  
**目标平台**: Windows / macOS / Linux

---

## 🎯 核心功能模块

| 功能模块 | 说明 |
|---------|------|
| 时间块规划 | 拖拽式时间块创建，专注单一任务 |
| 任务追踪 | 实际用时记录，计划 vs 执行对比 |
| 统计报表 | 日/周/月时间流向可视化 |
| 双线同步 | WebDAV + 云端账号双保险 |
| 离线优先 | 本地 SQLite 存储，有网自动同步 |
| 账号系统 | 用户注册/登录/数据隔离 |

### 成员E - WebDAV 同步 & 测试开发

**主要职责**:
- WebDAV 同步功能开发
- 数据同步算法
- 冲突检测与解决
- 测试与质量保证

**具体任务**:
1. WebDAV 客户端实现
```javascript
// webdav.js
const { createWebDAVClient } = require('webdav');

class WebDAVSync {
  constructor(config) {
    this.client = createWebDAVClient(config.url, {
      username: config.username,
      password: config.password
    });
    this.localDB = null;
  }
  
  // 上传本地数据到 WebDAV
  async uploadToCloud() {
    const data = await this.localDB.exportData();
    const encrypted = this.encrypt(data);
    await this.client.putFileContents('/timeblock/data.json', encrypted);
  }
  
  // 从 WebDAV 下载数据
  async downloadFromCloud() {
    const encrypted = await this.client.getFileContents('/timeblock/data.json');
    const data = this.decrypt(encrypted);
    return data;
  }
  
  // 双向同步 + 冲突解决
  async sync() {
    const localVersion = await this.getLocalVersion();
    const cloudVersion = await this.getCloudVersion();
    
    if (localVersion > cloudVersion) {
      await this.uploadToCloud();
    } else if (cloudVersion > localVersion) {
      await this.downloadFromCloud();
    } else {
      // 冲突检测与合并
      await this.mergeConflicts();
    }
  }
}
```
