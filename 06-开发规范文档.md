# 时间块记录器 - 开发规范文档

## 1. Git 规范

### 1.1 分支管理

| 分支名称 | 用途 | 说明 |
|----------|------|------|
| `main` | 主分支 | 生产环境代码，稳定版本 |
| `develop` | 开发分支 | 最新开发代码 |
| `feature/xxx` | 功能分支 | 新功能开发 |
| `bugfix/xxx` | Bug修复分支 | 修复Bug |
| `hotfix/xxx` | 紧急修复分支 | 生产环境紧急修复 |
| `release/xxx` | 发布分支 | 版本发布准备 |

### 1.2 分支创建流程

1. 从 `develop` 分支创建 `feature` 分支
2. 开发完成后，提交 PR 合并到 `develop`
3. 发布时，从 `develop` 创建 `release` 分支
4. 测试通过后，合并到 `main` 和 `develop`
5. 打 Tag 发布版本

### 1.3 Commit 规范

使用 Conventional Commits 规范：

```
<type>(<scope>): <subject>

<body>

<footer>
```

**Type 类型**:
- `feat`: 新功能
- `fix`: 修复Bug
- `docs`: 文档更新
- `style`: 代码格式调整
- `refactor`: 重构
- `test`: 测试相关
- `chore`: 构建/工具相关

**示例**:
```
feat(time-block): add drag and drop support

- Implement drag and drop for time blocks
- Add visual feedback during drag
- Update tests

Closes #123
```

### 1.4 PR 规范

- 每个 PR 对应一个 Issue
- PR 标题格式：`[Type] Description`
- PR 描述包含：改动内容、测试方法、关联 Issue
- 至少 1 人审核通过才能合并
- 合并前需要通过 CI 检查

### 3.3 Controller 规范

```typescript
import { Controller, Get, Post, Body, Param } from '@nestjs/common';
import { ApiTags, ApiOperation, ApiResponse } from '@nestjs/swagger';
import { TimeBlockService } from './time-block.service';
import { CreateTimeBlockDto } from './dto/create-time-block.dto';

@ApiTags('time-block')
@Controller('time-blocks')
export class TimeBlockController {
  constructor(private readonly timeBlockService: TimeBlockService) {}

  @Post()
  @ApiOperation({ summary: '创建时间块' })
  @ApiResponse({ status: 201, description: '创建成功' })
  create(@Body() createTimeBlockDto: CreateTimeBlockDto) {
    return this.timeBlockService.create(createTimeBlockDto);
  }

  @Get(':id')
  @ApiOperation({ summary: '获取时间块' })
  findOne(@Param('id') id: string) {
    return this.timeBlockService.findOne(id);
  }
}
```

### 3.4 Service 规范

```typescript
import { Injectable, NotFoundException } from '@nestjs/common';
import { InjectRepository } from '@nestjs/typeorm';
import { Repository } from 'typeorm';
import { TimeBlock } from './entities/time-block.entity';

@Injectable()
export class TimeBlockService {
  constructor(
    @InjectRepository(TimeBlock)
    private timeBlockRepository: Repository<TimeBlock>,
  ) {}

  async create(createDto: CreateTimeBlockDto): Promise<TimeBlock> {
    const timeBlock = this.timeBlockRepository.create(createDto);
    return this.timeBlockRepository.save(timeBlock);
  }

  async findOne(id: string): Promise<TimeBlock> {
    const timeBlock = await this.timeBlockRepository.findOneBy({ id });
    if (!timeBlock) {
      throw new NotFoundException(`TimeBlock #${id} not found`);
    }
    return timeBlock;
  }
}
```

## 4. 数据库规范

### 4.1 命名规范

- 表名：小写 + 下划线，如 `time_blocks`
- 字段名：小写 + 下划线，如 `start_time`
- 索引名：`idx_表名_字段名`，如 `idx_time_blocks_user_id`
- 外键名：`fk_表名_字段名`，如 `fk_time_blocks_category_id`

### 4.2 实体规范

```typescript
import { Entity, PrimaryGeneratedColumn, Column, ManyToOne } from 'typeorm';
import { Category } from './category.entity';

@Entity('time_blocks')
export class TimeBlock {
  @PrimaryGeneratedColumn('uuid')
  id: string;

  @Column({ type: 'varchar', length: 200 })
  title: string;

  @Column({ type: 'text', nullable: true })
  description?: string;

  @Column({ type: 'datetime', name: 'start_time' })
  startTime: Date;

  @Column({ type: 'datetime', name: 'end_time' })
  endTime: Date;

  @ManyToOne(() => Category, { onDelete: 'SET NULL' })
  category?: Category;

  @Column({ type: 'datetime', name: 'created_at', default: () => 'CURRENT_TIMESTAMP' })
  createdAt: Date;
}
```

## 6. 安全规范

### 6.1 认证授权

- 使用 JWT 进行身份认证
- Token 过期时间：24小时
- 敏感操作需要二次确认
- 实现 RBAC 权限控制

### 6.2 数据安全

- 密码使用 bcrypt 加密（salt rounds = 10）
- WebDAV 密码使用 AES-256 加密存储
- HTTPS 传输加密
- SQL 注入防护：使用 TypeORM 参数化查询
- XSS 防护：前端输出转义

### 6.3 API 安全

- 限流：每个用户每分钟最多 100 次请求
- 输入验证：使用 class-validator
- 错误处理：不泄露敏感信息
